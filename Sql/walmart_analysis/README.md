# Walmart Sales Analysis using SQL

## 📖 Project Overview

This project focuses on analyzing Walmart sales data using SQL. The objective is to explore the dataset, perform data cleaning, and answer real-world business questions using SQL queries. The project demonstrates practical SQL skills including data exploration, aggregation, filtering, grouping, conditional statements, date handling, and subqueries.

---

## 📊 Dataset Information

The dataset contains weekly sales information for multiple Walmart stores along with several economic and environmental factors.

**Dataset Features:**

- Store
- Date
- Weekly Sales
- Holiday Flag
- Temperature
- Fuel Price
- Consumer Price Index (CPI)
- Unemployment Rate

---

## 🎯 Objectives

- Import and explore the dataset.
- Perform data cleaning and preprocessing.
- Convert text dates into SQL DATE format.
- Analyze store performance.
- Compare holiday and non-holiday sales.
- Analyze sales trends over time.
- Study the effect of temperature, fuel price, CPI, and unemployment on sales.
- Solve business problems using SQL.
